## Apply

### Exercise 20

1.  Return to your data from the previous exercise (or start with the
    exercise solution)
2.  Fit a PCA model to the standardized predictors (not the outcome)
3.  Create a scree plot to choose $k$, the number of principal
    components to keep
4.  Using the chosen $k$, visualize a two-dimensional projection of the
    data
5.  Try and name the principal components based on the projection and
    their loadings
6.  Submit your code, output, and interpretation as a single PDF on
    Canvas
